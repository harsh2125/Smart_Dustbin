Section 01

What is the Smart Dustbin?

The Smart Dustbin is an Arduino-based automated waste segregation system. When you drop any waste item into the bin, the system automatically detects and classifies it as wet, dry, or metallic waste — then physically routes it into the correct compartment using a servo motor flap. No manual sorting required.

♻ Eco-Friendly ⚡ Real-Time Detection 🏠 Home & Office Ready 🦺 Hygienic — No Manual Sorting
Section 02

System Components

The system uses five core hardware components working together to detect, classify, and redirect waste automatically.

🧠

Arduino Uno

ATmega328P microcontroller. The central brain — reads all sensor data and controls the servo motor.

Central Controller
👁️

IR Sensor

Infrared sensor that detects when an object is placed in the bin opening. System activates only when needed.

Pin D2
💧

Rain / Moisture Sensor

Reads moisture content of the waste. If moisture exceeds threshold (20%), waste is classified as wet.

Pin A0
🔩

Metal Sensor

Digital sensor that detects the presence of metallic materials like aluminium foil, cans, and screws.

Pin D3
⚙️

Servo Motor

Rotates a physical flap 120°–240° to direct waste into the correct compartment.

Pin D9
Section 03

Quick Setup Guide

Follow these steps to get your Smart Dustbin running in minutes.

01

Connect Components

Wire the IR, rain, and metal sensors to Arduino pins D2, A0, D3. Connect servo to D9. Power sensors via VCC & GND.

02

Upload Code

Open Arduino IDE. Load the Smart Dustbin sketch. Select board as "Arduino Uno" and upload via USB.

03

Power On

Connect 5V USB power or external adapter. The servo will initialise and the IR sensor will begin scanning.

04

Position the Bin

Place the dustbin on a flat surface. Make sure the IR sensor faces the bin opening. All 3 compartments should be accessible.

05

Drop Waste

Simply drop waste into the top of the bin. The system detects, classifies, and sorts it automatically. No input needed.

06

Empty Compartments

Periodically empty each labelled compartment — Wet, Dry, and Metal — separately for proper recycling.

Section 04

How It Works — System Flow

Here is the complete decision flow the system follows every time waste is deposited.

1

Waste is Deposited

User drops any waste item into the top opening of the bin.

User Action
2

IR Sensor Activates (Pin D2)

The infrared sensor detects the presence of an object. If nothing is detected, the system stays idle and waits. This saves power.

Pin D2 — Digital
3

Arduino Reads Both Sensors

The Arduino Uno simultaneously reads the metal sensor (D3) and rain sensor (A0) to classify the waste type.

Arduino Uno
4

Metal Sensor Check (Pin D3)

If the metal sensor is triggered, the waste is immediately classified as metallic. The servo rotates 240° to the metal compartment.

Pin D3 — Digital
5

Rain / Moisture Sensor Check (Pin A0)

If the moisture reading exceeds 20%, the waste is classified as wet. The servo rotates 120° to the wet compartment.

Pin A0 — Analog
6

Default → Dry Waste

If neither sensor is triggered, the item defaults to dry waste. The flap opens straight into the dry compartment.

Default Case
7

Servo Motor Directs Waste (Pin D9)

The servo motor rotates the internal flap to the classified compartment, deposits the waste, and returns to the neutral position ready for the next item.

Pin D9 — PWM

Section 07

Tips, Notes & Warnings

✅

Calibrate the Rain Sensor First

Before regular use, test the rain sensor threshold. The default threshold is 20% — adjust in code if your sensor reads differently. Use a slightly damp cloth for calibration.

⚠️

Keep Rain Sensor Dry Between Uses

The moisture sensor should only be exposed briefly when waste is dropped. Prolonged moisture exposure can damage the sensor. Ensure the sensor is in a protected position.

💡

One Item at a Time

Drop waste one item at a time for accurate classification. Multiple simultaneous items can confuse the sensors and lead to incorrect routing.

🔋

Stable Power Supply Required

Use a 5V/2A power adapter or USB connection from a computer. Unstable power can cause servo jitter or sensor misreads.

🧹

Clean Sensors Weekly

Dust or residue on the IR sensor or metal sensor can cause false triggers. Wipe gently with a dry cloth each week.
